## Description
A visualization package for dynamical systems

## Installation

Requires Julia >1.1 and install package below
```julia
using Pkg
Pkg.add("GR")
Pkg.add("Plots")
Pkg.add("FFMPEG")
Pkg.add("https://github.com/HaoLi111/VisDyn.jl")# finally install this package
```

## Usage
```julia
using VisDyn
```
